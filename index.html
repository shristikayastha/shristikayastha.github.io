Fake News Detection Model

GitHub ↗
Problem

The rapid spread of misinformation on social media platforms has made it difficult to distinguish between real and fake news. This project focuses on building a machine learning model that can automatically classify news articles as genuine or fake.

My Contribution

Collected and preprocessed a dataset containing over 40,000 news articles. Implemented NLP techniques including text cleaning, tokenisation, and TF-IDF vectorisation. 02

Devanagari Handwritten Digit Recognition System

GitHub ↗
Problem

Handwritten digit recognition in the Devanagari script is challenging due to variations in handwriting styles, limited datasets, and preprocessing requirements. Most OCR systems focus on Latin scripts, leaving Nepali and Devanagari numerals less explored. This project aims to build a system capable of accurately recognizing Devanagari handwritten digits.

My Contribution

Built a Devanagari Handwritten Digit Recognition system using deep learning and transfer learning with a pretrained ResNet18 model.Preprocessed images by performing grayscale conversion, resizing, normalization, and splitting data into training and testing sets.Fine-tuned the ResNet18 model to handle single-channel images and classify 10 digits, using Cross-Entropy Loss and Adam optimizer.Evaluated the system using metrics such as accuracy, precision, recall, F1-score, and visualized performance with a confusion matrix.Achieved 99% training accuracy and 98.09% test accuracy, demonstrating robust digit recognition performance.

Python PyTorch scikit-learn Torchvision Kaggledatasets Matplotlib
